Amazon Sales Analysis

In this project, I analyzed Amazon sales data containing 1,23,000+ records to understand sales trends, customer behavior, and regional performance.I performed data cleaning in Excel, wrote SQL queries for analysis, and built an interactive dashboard in Power BI.

- SQL (Data Analysis)
- Power BI (Dashboard & Visualization)

--> Key Insights
- KPI metrics (Sales, Orders, Quantity)
- Sales trend analysis
- Category performance analysis
- Geographic sales distribution
- Interactive filters (slicers)
