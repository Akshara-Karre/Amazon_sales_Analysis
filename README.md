# Amazon Sales Analysis

## 📊 Project Overview
This project analyzes Amazon sales data to identify key business insights such as top-performing categories, sales trends, and regional performance.

## 🛠 Tools Used
- Excel (Data Cleaning)
- SQL (Data Analysis)
- Power BI (Dashboard & Visualization)

## 📈 Key Insights
- KPI metrics (Sales, Orders, Quantity)
- Sales trend analysis
- Category performance analysis
- Geographic sales distribution
- Interactive filters (slicers)

## 📂 Files Included
- Amazon_Sales_Dashboard.pbix
- Amazon_Sales_Report.pdf
- Amazon_Sales_SQL_Queries.sql

## 🚀 Outcome
Built an interactive dashboard to help understand sales performance and support business decisions.
